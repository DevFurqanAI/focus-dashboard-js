# 🌅 Focus Dashboard

A minimal yet interactive landing page that adapts to you throughout the day.  
It displays real-time updates, adjusts visuals based on time, and keeps track of your daily focus — all in a clean, distraction-free interface.

---

## ✨ Features

- ⏰ Live clock (updates every second)
- 🌄 Dynamic background based on time of day
- 👤 Editable user name (auto-saved)
- 🎯 Daily focus input (persistent)
- 💬 Random motivational quote
- 💾 Data stored using localStorage
- 📱 Responsive and centered layout
- 🎨 Smooth background transitions

---

## 🛠️ Tech Stack

- HTML5  
- CSS3 (Flexbox, Responsive Design, Transitions)  
- JavaScript (DOM Manipulation, Events, Local Storage)
